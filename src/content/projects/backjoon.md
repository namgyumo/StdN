## 개요

백준 온라인 저지에서 알고리즘 문제를 꾸준히 풀고 기록하는 저장소. 실버 레벨을 중심으로 정렬, 탐색, 동적 프로그래밍, 그래프 등 핵심 알고리즘을 C 언어로 직접 구현했다.

## 주요 카테고리

- 정렬 (버블, 선택, 삽입, 퀵, 병합)
- 완전탐색 / 백트래킹
- 동적 프로그래밍 (DP)
- 그래프 탐색 (BFS / DFS)
- 문자열 처리

## 학습 방향

단순히 정답을 맞히는 것보다 시간복잡도와 공간복잡도를 의식하며 풀이를 작성하고 있다. 같은 문제를 다른 접근 방식으로 풀어보는 과정에서 알고리즘 선택의 tradeoff를 직접 체감하는 것이 목표.

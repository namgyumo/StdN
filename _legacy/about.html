VALUES

성능과 효율을 가장 중요하게 생각합니다. 불필요한 리소스 낭비가 눈에 밟히는 편이고, 사용자가 느끼는 속도 차이를 코드 수준에서 만들어내는 것에 관심이 있습니다.

GOALS

졸업 후에도 계속 공부하면서, 시스템 소프트웨어 분야에서 실력 있는 개발자로 자리잡는 게 목표입니다. 화려한 기술보다 구조를 제대로 이해하는 개발자가 되고 싶습니다.

MOTIVE

C언어로 알고리즘 문제를 풀다가, 로직 하나 바꿨을 뿐인데 실행 시간이 눈에 띄게 달라지는 걸 보고 알고리즘과 자료구조에 빠졌습니다. 포인터로 메모리를 직접 다루는 것도 그때부터 재미있어졌고, 시스템 밑단에서 데이터가 어떻게 움직이는지 이해하는 게 아직도 즐겁습니다.

INTERESTS

알고리즘과 자료구조의 원리를 파고드는 게 재미있습니다. 특히 포인터와 메모리 구조를 이해하고 그걸 코드로 직접 구현해보는 것, 온라인 저지 문제 푸는 것을 즐겨 합니다.

HOBBIES

평소 코딩 외의 시간에는 서브컬쳐 게임을 즐기거나 음악을 즐겨 듣습니다. 뮤지컬도 가끔 보러 갑니다.
